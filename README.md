## ✨ Key Features
* **🔐 Authentication System:** Secure User Registration (`Register`) and Login (`Login`) mechanisms driven by PHP sessions.
* **📊 Dynamic Dashboard:** Personalized "Welcome" greetings fetched from user profiles alongside an overview of the content.
* **📂 Category Management:** Full capability to Add, View, and organize news categories (e.g., Sports News, Political News, etc.).
* **📝 News CRUD Operation:** Add and display news articles with custom titles, assigned categories, and dynamic image upload features.
* **🗑️ Soft Delete Archive:** Dedicated section for deleted news records allowing easy review or management.
* **📱 Responsive Design:** Sleek and clean interface fully optimized for mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack
* **🎨 Frontend:** HTML5, CSS3, Bootstrap 5 (For styling and layout responsiveness).
* **⚙️ Backend:** PHP (For server-side logic, form handling, and session tracking).
* **🗄️ Database:** MySQL (For relational data storage and table connections).
* **🌐 Hosting:** InfinityFree (Live production server and remote MySQL server hosting).

## 💻 Local Installation & Setup
To run this project locally using a local server environment like XAMPP:

1. **Clone the repository** or download the source files.
2. Move the project folder to your local root directory (e.g., `C:/xampp/htdocs/`).
3. Open **phpMyAdmin** and create a new database.
4. **Import** the included `.sql` database file into your newly created database.
5. Update your database configuration settings inside `db.php` (Server: `localhost`, User: `root`, Password: `""`).
6. Open your web browser and navigate to:
   `http://localhost/YOUR_PROJECT_FOLDER_NAME/login.php`
